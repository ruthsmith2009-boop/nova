🎙 Vapi AI (Voice Brain)
How to connect:
1. Sign up at vapi.ai
2. Dashboard → API Keys → copy your key
3. Buy/import a phone number → copy its ID
4. Paste both into ~/aria-re/.env:
VAPI_API_KEY=...
VAPI_PHONE_NUMBER_ID=...
5. Restart the backend

Vapi runs the live AI conversation using Claude. It handles speaking, listening, objections, and asking for the decision maker.

☎ Twilio (Phone Line)
How to connect:
1. Sign up at twilio.com, add credit
2. Buy a local number (408 area code)
3. Console → copy Account SID + Auth Token
4. Register A2P 10DLC (avoids spam flags)
5. Paste into ~/aria-re/.env and restart

You can connect Twilio directly to Vapi inside the Vapi dashboard — that's the simplest path. NOVA works either way.

🔗 Webhook (for call results)

For NOVA to receive call results automatically, Vapi needs a public URL. For local testing, run ngrok http 8000 and set PUBLIC_BASE_URL in .env to the ngrok URL.

📘 Facebook & Instagram

One Meta token connects both. Requires a Facebook Business Page + Instagram Business Account.

How to get your token:
1. Go to developers.facebook.com
2. Create an App → Add "Pages" product
3. Graph API Explorer → generate Page Access Token
4. Paste below and save to your .env file
🐦 Twitter / X
How to get your keys:
1. Go to developer.twitter.com
2. Create a project + app
3. Keys and Tokens tab → generate all 4 keys
💼 LinkedIn
How to get your token:
1. Go to linkedin.com/developers
2. Create an App → request "Share on LinkedIn" permission
3. OAuth 2.0 → generate access token
